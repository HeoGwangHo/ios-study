# MapView를 이용하여 현재 위치 보여주기<br>
### 주요기능<br>
**1.[현재 위치]를 선택하면 ios시뮬레이터에서 미리 설정해둔 지역이 표시 됨**<br>
**2.다른 메뉴를 선택하면 미리 설정해둔 위이세 핀이 설치 됨**<br>
**3.핀을 클릭하면 설명문이 나타남**<br>
